# Peg Solitaire Game (CS449 Project)

## Description
This project is a Peg Solitaire game developed in Python for CS449.  
The program includes a graphical user interface (GUI) built with Tkinter and a separate game logic class.

The game allows the player to select different board types and make moves by clicking pegs on the board.

## Features
- Choose board type:
  - English
  - Hexagon
  - Diamond
- Start a new game
- Make moves by clicking pegs
- Detect when the game is over
- Automated unit tests using Python unittest
